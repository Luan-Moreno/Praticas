# Práticas de Excel

Esta pasta contém exercícios e práticas focadas em melhorar minhas habilidades no Excel.

## Conteúdo

- **Funções**: Utilização de funções como PROCV, SOMASES, BDMEDIA, SEERRO, etc.
- **Tabelas Dinâmicas e Dashboards**: Criação e customização de tabelas dinâmicas. Criação de diferentes tipos de dashboards para visualização de dados.
- **Modelagem de dados com Power Pivot**: Criação de relacionamentos entre tabelas (Modelos de Dados) e Medidas utilizando DAX. Aplicação em Tabelas Dinâmicas e Dashboards para análises avançadas.
- **Transformação de dados com Power Query**: Importação de dados de diversas fontes, seguida de limpeza, mescla e combinação. Utilização em Tabelas Dinâmicas e Dashboards para visualizações atualizadas.
- **Macros**: Automação de atividades com macros.
